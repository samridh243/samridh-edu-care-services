# Samridh Edu Care Services

A modern, responsive educational consultancy website built with **React**, **Vite**, and **Tailwind CSS v4**.

The website is designed for **Samridh Edu Care Services**, providing students with information about admissions, career guidance, scholarship assistance, NIOS, diploma, degree, postgraduate, and skill-based courses.

---

## Features

* Modern responsive design
* Mobile-friendly navigation
* Animated hero section
* Course filtering
* Admission inquiry section
* Scholarship information
* Career guidance
* FAQ section
* Testimonials
* Smooth animations using Motion
* Professional UI with Tailwind CSS
* Optimized for desktop, tablet, and mobile devices

---

## Tech Stack

* React
* Vite
* JavaScript
* Tailwind CSS v4
* Motion for React
* Lucide React Icons

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate into the project:

```bash
cd samridh-edu-care-services
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

---

## Customization

### Logo

Replace the existing logo by updating:

```text
src/assets/logo.png
```

---

### Contact Information

Update the contact details inside:

```text
src/data.json
```

---

### Courses

Modify or add courses in:

```text
src/data.js
```

---

### Hero Images

Replace the images inside the assets/public images folder while keeping the same filenames to avoid changing component code.
